1)как создать свою виртуальную среду
python -m venv название_виртуальной_среды
2)как активировать виртуальную среду
windows
название_виртуальной_среды\scripts\activate
ubuntu/linux
source название_виртуальной_среды/bin/activate
3) как установить библиотеку [2 этап обязателен]
pip install название_библиотеки
4) как выйти из среды
deactivate
5) как получить список библиотек которые были использованы
pip freeze > req.txt
6) чтобы установить все библиотеки из другой виртуальной среды в новую необходимо[5]
а затем воспользоваться командой
pip install -r req.txt
